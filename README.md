# 0verDistribution


## Install
```
git clone https://github.com/0verseas/0verDistribution.git
cd 0verDistribution
npm install
cp src/js/env.js.example src/js/env.js
```
edit the config file in `src/env.js`

## Run
```
npm run serve
```

## Deploy
```
npm run build
```
the built static files will be in the `dist`
